\doxysubsection{Detailed Description}
An immutable object representing a geographical point in \doxylink{classfirebase_1_1firestore_1_1_firestore}{Firestore}. The point is represented as a latitude/longitude pair.

Latitude values are in the range of \mbox{[}-\/90, 90\mbox{]}. Longitude values are in the range of \mbox{[}-\/180, 180\mbox{]}.

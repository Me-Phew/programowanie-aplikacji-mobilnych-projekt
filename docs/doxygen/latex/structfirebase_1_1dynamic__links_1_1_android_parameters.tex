Android Parameters. 

Note that the strings used by the struct are not copied, as so must either be statically allocated, or must persist in memory until the \doxylink{structfirebase_1_1dynamic__links_1_1_dynamic_link_components}{Dynamic\+Link\+Components} that uses them goes out of scope.

\doxysubsection{Detailed Description}
Value listener interface. Subclasses of this listener class can be used to receive events about data changes at a location. Attach the listener to a location using \doxylink{classfirebase_1_1database_1_1_query_aeb583c0c607b01692f9aaa9a59432810}{Database\+Reference\+::\+Add\+Value\+Listener()} or \doxylink{classfirebase_1_1database_1_1_query_aeb583c0c607b01692f9aaa9a59432810}{Query\+::\+Add\+Value\+Listener()}, and \doxylink{classfirebase_1_1database_1_1_value_listener_a02ce2f03e20564a124b5d290158494a8}{On\+Value\+Changed()} will be called once immediately, and again when the value changes.

\doxysubsection{Detailed Description}
Loads and displays Google Mobile Ads \doxylink{classfirebase_1_1gma_1_1_ad_view}{Ad\+View} ads. 

Each \doxylink{classfirebase_1_1gma_1_1_ad_view}{Ad\+View} object corresponds to a single GMA ad placement of a specified size. There are methods to load an ad, move it, show it and hide it, and retrieve the bounds of the ad onscreen.

\doxylink{classfirebase_1_1gma_1_1_ad_view}{Ad\+View} objects provide information about their current state through Futures. Methods like \doxylink{classfirebase_1_1gma_1_1_ad_view_a5abfe11cf329ae36ea6f2a1207bb8efe}{Initialize}, \doxylink{classfirebase_1_1gma_1_1_ad_view_aad082060ac538b3e5073765749e8e843}{Load\+Ad}, and \doxylink{classfirebase_1_1gma_1_1_ad_view_aa1f18f94b19a06cf7eb2822e71fd8379}{Hide} each have a corresponding \doxylink{classfirebase_1_1_future}{Future} from which the result of the last call can be determined. The two variants of \doxylink{classfirebase_1_1gma_1_1_ad_view_ad3036c54c3d364cc61b4dcef7efe1073}{Set\+Position} share a single result \doxylink{classfirebase_1_1_future}{Future}, since they\textquotesingle{}re essentially the same action.

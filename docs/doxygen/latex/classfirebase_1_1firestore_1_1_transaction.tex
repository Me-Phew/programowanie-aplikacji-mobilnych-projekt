\doxysubsection{Detailed Description}
\doxylink{classfirebase_1_1firestore_1_1_transaction}{Transaction} provides methods to read and write data within a transaction. 

You cannot create a {\ttfamily \doxylink{classfirebase_1_1firestore_1_1_transaction}{Transaction}} directly; use {\ttfamily \doxylink{classfirebase_1_1firestore_1_1_firestore_a334fd6ca23f019c97229926986a59e4a}{Firestore\+::\+Run\+Transaction()}} function instead.

\begin{DoxyNote}{Note}
\doxylink{classfirebase_1_1firestore_1_1_firestore}{Firestore} classes are not meant to be subclassed except for use in test mocks. Subclassing is not supported in production code and new SDK releases may break code that does so. 
\end{DoxyNote}

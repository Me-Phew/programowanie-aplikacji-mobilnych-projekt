Sets an approximate cache size threshold for the on-\/disk data. If the cache grows beyond this size, Cloud \doxylink{classfirebase_1_1firestore_1_1_firestore}{Firestore} will start removing data that hasn\textquotesingle{}t been recently used. The size is not a guarantee that the cache will stay below that size, only that if the cache exceeds the given size, cleanup will be attempted.

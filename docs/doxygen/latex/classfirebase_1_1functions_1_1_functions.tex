\doxysubsection{Detailed Description}
Entry point for the Firebase C++ SDK for Cloud \doxylink{classfirebase_1_1functions_1_1_functions}{Functions}. 

To use the SDK, call \doxylink{classfirebase_1_1functions_1_1_functions_a7890209290654e12353dc67b1dc31a96}{firebase\+::functions\+::\+Functions\+::\+Get\+Instance()} to obtain an instance of \doxylink{classfirebase_1_1functions_1_1_functions}{Functions}, then use \doxylink{classfirebase_1_1functions_1_1_functions_afae54655466b14c27e3d6da35c81bace}{Get\+Https\+Callable()} to obtain references to callable functions. From there you can call them with Callable\+Reference\+::\+Call().

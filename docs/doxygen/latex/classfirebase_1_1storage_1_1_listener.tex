\doxysubsection{Detailed Description}
Base class used to receive pause and progress events on a running read or write operation. 

Subclasses of this listener class can be used to receive events about data transfer progress a location. Attach the listener to a location using \doxylink{classfirebase_1_1storage_1_1_storage_reference_a606eae171fe4dda094bdafde307e5022}{Storage\+Reference\+::\+Get\+Bytes()}, \doxylink{classfirebase_1_1storage_1_1_storage_reference_a77102fc20b083b13a3e9482d1b4b4678}{Storage\+Reference\+::\+Get\+File()}, \doxylink{classfirebase_1_1storage_1_1_storage_reference_a3da5eae65c4858dd68281f69bc1239a7}{Storage\+Reference\+::\+Put\+Bytes()}, and \doxylink{classfirebase_1_1storage_1_1_storage_reference_a1c1990b1936b13337ac55aa26f40db93}{Storage\+Reference\+::\+Put\+File()}; then \doxylink{classfirebase_1_1storage_1_1_listener_a002fd09bad9d749588407387bc7a0527}{On\+Paused()} will be called whenever the Read or Write operation is paused, and \doxylink{classfirebase_1_1storage_1_1_listener_a3049e5cceb71b018b34d9170e4948fef}{On\+Progress()} will be called periodically as the transfer makes progress.

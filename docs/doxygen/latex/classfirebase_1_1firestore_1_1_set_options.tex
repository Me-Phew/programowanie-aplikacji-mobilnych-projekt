\doxysubsection{Detailed Description}
An options object that configures the behavior of Set() calls. 

By providing the \doxylink{classfirebase_1_1firestore_1_1_set_options}{Set\+Options} objects returned by \doxylink{classfirebase_1_1firestore_1_1_set_options_a64fc825a4de04509f392f3947075cc14}{Merge()}, the Set() methods in \doxylink{classfirebase_1_1firestore_1_1_document_reference}{Document\+Reference}, \doxylink{classfirebase_1_1firestore_1_1_write_batch}{Write\+Batch} and \doxylink{classfirebase_1_1firestore_1_1_transaction}{Transaction} can be configured to perform granular merges instead of overwriting the target documents in their entirety.

\doxysubsection{Detailed Description}
Allows you to register server-\/side actions to occur when the client disconnects. Each method you call (with the exception of Cancel) will queue up an action on the data that will be performed by the server in the event the client disconnects. To reset this queue, call \doxylink{classfirebase_1_1database_1_1_disconnection_handler_a1a3378d87dc8cd37c9817b019e7ec6d8}{Cancel()}.

A \doxylink{classfirebase_1_1database_1_1_disconnection_handler}{Disconnection\+Handler} is associated with a specific location in the database, as they are obtained by calling \doxylink{classfirebase_1_1database_1_1_database_reference_a0eee7be37c0aeac0172356f0e46f112b}{Database\+Reference\+::\+On\+Disconnect()}.

\doxysubsection{Detailed Description}
A \doxylink{classfirebase_1_1firestore_1_1_document_snapshot}{Document\+Snapshot} contains data read from a document in your \doxylink{classfirebase_1_1firestore_1_1_firestore}{Firestore} database. 

The data can be extracted with the \doxylink{classfirebase_1_1firestore_1_1_document_snapshot_a27c719118b16ca67f96634015863baa9}{Get\+Data()} method, or by using \doxylink{classfirebase_1_1firestore_1_1_document_snapshot_a726989caf1c55d7d355af3d5bfa3f518}{Get()} to access a specific field. For a \doxylink{classfirebase_1_1firestore_1_1_document_snapshot}{Document\+Snapshot} that points to a non-\/existing document, any data access will cause a failed assertion. You can use the \doxylink{classfirebase_1_1firestore_1_1_document_snapshot_a0e3485a381ed63afa91a712b3b5015a9}{exists()} method to explicitly verify a document\textquotesingle{}s existence.

\begin{DoxyNote}{Note}
\doxylink{classfirebase_1_1firestore_1_1_firestore}{Firestore} classes are not meant to be subclassed except for use in test mocks. Subclassing is not supported in production code and new SDK releases may break code that does so. 
\end{DoxyNote}

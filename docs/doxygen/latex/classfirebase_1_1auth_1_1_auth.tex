Registers a listener to changes in the authentication state. 

There can be more than one listener registered at the same time. The listeners are called asynchronously, possibly on a different thread.

Authentication state changes are\+:
\begin{DoxyItemize}
\item Right after the listener has been registered
\item When a user signs in
\item When the current user signs out
\item When the current user changes
\end{DoxyItemize}

It is a recommended practice to always listen to sign-\/out events, as you may want to prompt the user to sign in again and maybe restrict the information or actions they have access to.

Use Remove\+Auth\+State\+Listener to unregister a listener.

\begin{DoxyNote}{Note}
The caller owns {\ttfamily listener} and is responsible for destroying it. When {\ttfamily listener} is destroyed, or when \doxylink{classfirebase_1_1auth_1_1_auth}{Auth} is destroyed, Remove\+Auth\+State\+Listener is called automatically. 
\end{DoxyNote}

Inequality operator, only meant for internal use. 

Explanation\+: In order to use \doxylink{classfirebase_1_1_variant}{Variant} as a key for std\+::map, we must provide a comparison function. This comparison function is ONLY for std\+::map to be able to use a \doxylink{classfirebase_1_1_variant}{Variant} as a map key.

We define v1 \texorpdfstring{$<$}{<} v2 IFF\+:
\begin{DoxyItemize}
\item If different types, compare type as int\+: v1.\+type() \texorpdfstring{$<$}{<} v2.\+type() (note\+: this means that Variant(1) \texorpdfstring{$<$}{<} \doxylink{classfirebase_1_1_variant}{Variant}(0.\+0) -\/ be careful!)
\item If both are int64\+: v1.\+int64\+\_\+value() \texorpdfstring{$<$}{<} v2.\+int64\+\_\+value();
\item If both are double\+: v1.\+double\+\_\+value() \texorpdfstring{$<$}{<} v2.\+double\+\_\+value()
\item If both are bool\+: v1.\+bool\+\_\+value() \texorpdfstring{$<$}{<} v2.\+bool\+\_\+value();
\item If both are either static or mutable strings\+: strcmp(v1, v2) \texorpdfstring{$<$}{<} 0
\item If both are vectors\+:
\begin{DoxyItemize}
\item If v1\mbox{[}0\mbox{]} \texorpdfstring{$<$}{<} v2\mbox{[}0\mbox{]}, that means v1 \texorpdfstring{$<$}{<} v2 == true. Otherwise\+:
\item If v1\mbox{[}0\mbox{]} \texorpdfstring{$>$}{>} v2\mbox{[}0\mbox{]}, that means v1 \texorpdfstring{$<$}{<} v2 == false. Otherwise\+:
\item Continue to the next element of both vectors and compare again.
\item If you reach the end of one vector first, that vector is considered to be lesser.
\end{DoxyItemize}
\item If both are maps, iterate similar to vectors (since maps are ordered), but for each element, first compare the key, then the value.
\item If both are blobs, the smaller-\/sized blob is considered lesser. If both blobs are the same size, use memcmp to compare the bytes.
\end{DoxyItemize}

We have defined this operation such that if !(v1 \texorpdfstring{$<$}{<} v2) \&\& !(v2 \texorpdfstring{$<$}{<} v1), it must follow that v1 == v2.

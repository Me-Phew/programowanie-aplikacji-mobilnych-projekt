\doxysubsection{Detailed Description}
Entry point for the Firebase C++ SDK for Remote Config. 

To use the SDK, call \doxylink{classfirebase_1_1remote__config_1_1_remote_config_ae09919cd51bac22b4bf5fb91e4b07505}{firebase\+::remote\+\_\+config\+::\+Remote\+Config\+::\+Get\+Instance()} to obtain an instance of \doxylink{classfirebase_1_1remote__config_1_1_remote_config}{Remote\+Config}, then call operations on that instance. The instance contains the complete set of FRC parameter values available to your app. The instance also stores values fetched from the FRC Server until they are made available for use with a call to \doxylink{classfirebase_1_1remote__config_1_1_remote_config_a699b15146fb4d9570513c65e135d5cb5}{Activate()}.

Asynchronously fetches and then activates the fetched configs.

If the time elapsed since the last fetch from the Firebase Remote Config backend is more than the default minimum fetch interval, configs are fetched from the backend.

After the fetch is complete, the configs are activated so that the fetched key value pairs take effect.

\begin{DoxyReturn}{Returns}
A \doxylink{classfirebase_1_1_future}{Future} that contains true if the current call activated the fetched configs; if no configs were fetched from the backend and the local fetched configs have already been activated, the future will contain false. 
\end{DoxyReturn}

The type of each \doxylink{classfirebase_1_1_variant}{Variant} in the map determines the type of data for which you are providing a default. For example, boolean values can be retrieved with Get\+Bool(), integer values can be retrieved with \doxylink{classfirebase_1_1remote__config_1_1_remote_config_a1b376e125df82c8bc61b6e4686896122}{Get\+Long()}, double values can be retrieved with \doxylink{classfirebase_1_1remote__config_1_1_remote_config_aeb6dc379aef6243e944e6592b4a34ffc}{Get\+Double()}, string values can be retrieved with \doxylink{classfirebase_1_1remote__config_1_1_remote_config_a2a0912229bc6bede3dd11cfe381f7886}{Get\+String()}, and binary data can be retrieved with \doxylink{classfirebase_1_1remote__config_1_1_remote_config_af544c66efdec38ad1d4c5f5870476e85}{Get\+Data()}.

\doxysubsection{Detailed Description}
A \doxylink{classfirebase_1_1database_1_1_data_snapshot}{Data\+Snapshot} instance contains data from a Firebase \doxylink{classfirebase_1_1database_1_1_database}{Database} location. Any time you read \doxylink{classfirebase_1_1database_1_1_database}{Database} data, you receive the data as a \doxylink{classfirebase_1_1database_1_1_data_snapshot}{Data\+Snapshot}. These are efficiently-\/generated and cannot be changed. To modify data, use \doxylink{classfirebase_1_1database_1_1_database_reference_a03ec27c242b8692f1dc2d131bbc4b205}{Database\+Reference\+::\+Set\+Value()} or \doxylink{classfirebase_1_1database_1_1_database_reference_a15b5a1434665fe3f13c9ce4a649b09b0}{Database\+Reference\+::\+Run\+Transaction()}.

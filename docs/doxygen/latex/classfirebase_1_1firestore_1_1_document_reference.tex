\doxysubsection{Detailed Description}
A \doxylink{classfirebase_1_1firestore_1_1_document_reference}{Document\+Reference} refers to a document location in a \doxylink{classfirebase_1_1firestore_1_1_firestore}{Firestore} database and can be used to write, read, or listen to the location. 

There may or may not exist a document at the referenced location. A \doxylink{classfirebase_1_1firestore_1_1_document_reference}{Document\+Reference} can also be used to create a \doxylink{classfirebase_1_1firestore_1_1_collection_reference}{Collection\+Reference} to a subcollection.

Create a \doxylink{classfirebase_1_1firestore_1_1_document_reference}{Document\+Reference} via {\ttfamily \doxylink{classfirebase_1_1firestore_1_1_firestore_a63e7155867ce6daf744c1738e029492a}{Firestore\+::\+Document}(const std\+::string\& path)}.

\begin{DoxyNote}{Note}
\doxylink{classfirebase_1_1firestore_1_1_firestore}{Firestore} classes are not meant to be subclassed except for use in test mocks. Subclassing is not supported in production code and new SDK releases may break code that does so. 
\end{DoxyNote}

\doxysubsection{Detailed Description}
Authenticates with Federated OAuth Providers via the \doxylink{classfirebase_1_1auth_1_1_auth}{firebase\+::auth\+::\+Auth} and \doxylink{classfirebase_1_1auth_1_1_user}{firebase\+::auth\+::\+User} classes. 

Once configured with a provider id, and with OAuth scope and OAuth custom parameters via an Fedearted\+OAuth\+Provider\+Data structure, an object of this class may be used via \doxylink{classfirebase_1_1auth_1_1_auth_a6894064bd6e3804dee7d86786e93680f}{Auth\+::\+Sign\+In\+With\+Provider} to sign-\/in users, or via \doxylink{classfirebase_1_1auth_1_1_user_a0ab32323766476a0d16afd7f4b0074d4}{User\+::\+Link\+With\+Provider} and \doxylink{classfirebase_1_1auth_1_1_user_a35603c0b5336c919cbf90402959dfcd2}{User\+::\+Reauthenticate\+With\+Provider} for cross account linking and user reauthentication, respectively.

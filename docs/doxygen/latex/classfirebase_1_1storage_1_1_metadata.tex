\doxysubsection{Detailed Description}
\doxylink{classfirebase_1_1storage_1_1_metadata}{Metadata} stores default attributes such as size and content type. 

\doxylink{classfirebase_1_1storage_1_1_metadata}{Metadata} for a \doxylink{classfirebase_1_1storage_1_1_storage_reference}{Storage\+Reference}. You may also store custom metadata key value pairs. \doxylink{classfirebase_1_1storage_1_1_metadata}{Metadata} values may be used to authorize operations using declarative validation rules.

\doxysubsection{Detailed Description}
Implementation of an {\ttfamily \doxylink{classfirebase_1_1app__check_1_1_app_check_provider_factory}{App\+Check\+Provider\+Factory}} that builds Debug\+App\+Check\+Providers.

Debug\+App\+Check\+Provider can exchange a debug token registered in the Firebase console for a Firebase \doxylink{classfirebase_1_1_app}{App} Check token. The debug provider is designed to enable testing applications on a simulator or test environment.

NOTE\+: Do not use the debug provider in applications used by real users.

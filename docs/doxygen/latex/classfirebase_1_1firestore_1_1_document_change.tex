\doxysubsection{Detailed Description}
A \doxylink{classfirebase_1_1firestore_1_1_document_change}{Document\+Change} represents a change to the documents matching a query. 

\doxylink{classfirebase_1_1firestore_1_1_document_change}{Document\+Change} contains the document affected and the type of change that occurred (added, modified, or removed).

\begin{DoxyNote}{Note}
\doxylink{classfirebase_1_1firestore_1_1_firestore}{Firestore} classes are not meant to be subclassed except for use in test mocks. Subclassing is not supported in production code and new SDK releases may break code that does so. 
\end{DoxyNote}

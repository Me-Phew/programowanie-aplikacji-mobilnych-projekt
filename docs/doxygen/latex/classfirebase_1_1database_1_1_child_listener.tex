\doxysubsection{Detailed Description}
Child listener interface. Subclasses of this listener class can be used to receive events about changes in the child locations of a \doxylink{classfirebase_1_1database_1_1_query}{firebase\+::database\+::\+Query} or \doxylink{classfirebase_1_1database_1_1_database_reference}{firebase\+::database\+::\+Database\+Reference}. Attach the listener to a location with \doxylink{classfirebase_1_1database_1_1_query_a6ac1ee1619acbc89b03824e562cc9750}{Query\+::\+Add\+Child\+Listener()} or \doxylink{classfirebase_1_1database_1_1_query_a6ac1ee1619acbc89b03824e562cc9750}{Database\+Reference\+::\+Add\+Child\+Listener()} and the appropriate method will be triggered when changes occur.

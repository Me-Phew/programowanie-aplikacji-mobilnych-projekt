\doxysubsection{Detailed Description}
\doxylink{class_event}{Event} parameter. 

Parameters supply information that contextualize events (see \doxylink{namespacefirebase_1_1analytics_a93b61cf7740883d81eeaa442d951ac82}{Log\+Event}). You can associate up to 25 unique Parameters with each event type (name).

$<$\+SWIG$>$  $<$/\+SWIG$>$ 

\doxylink{structfirebase_1_1analytics_1_1_parameter}{Parameter} names must be a combination of letters and digits (matching the regular expression \mbox{[}a-\/z\+A-\/\+Z0-\/9\mbox{]}) between 1 and 40 characters long starting with a letter \mbox{[}a-\/z\+A-\/Z\mbox{]} character. The "{}firebase\+\_\+"{}, "{}google\+\_\+"{} and "{}ga\+\_\+"{} prefixes are reserved and should not be used.

\doxylink{structfirebase_1_1analytics_1_1_parameter}{Parameter} string values can be up to 100 characters long.

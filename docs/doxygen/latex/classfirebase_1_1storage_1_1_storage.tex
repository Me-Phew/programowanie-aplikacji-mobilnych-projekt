\doxysubsection{Detailed Description}
Entry point for the Firebase C++ SDK for Cloud \doxylink{classfirebase_1_1storage_1_1_storage}{Storage}. 

To use the SDK, call \doxylink{classfirebase_1_1storage_1_1_storage_a057fc8bc69877c7c674f4b220dba1f9f}{firebase\+::storage\+::\+Storage\+::\+Get\+Instance()} to obtain an instance of \doxylink{classfirebase_1_1storage_1_1_storage}{Storage}, then use \doxylink{classfirebase_1_1storage_1_1_storage_a476a4580fbe47370076974dd442ed8e0}{Get\+Reference()} to obtain references to child blobs. From there you can upload data with Storage\+Reference\+::\+Put\+Stream(), get data via Storage\+Reference\+::\+Get\+Stream().

\doxysubsection{Detailed Description}
Interface for a provider that generates {\ttfamily \doxylink{structfirebase_1_1app__check_1_1_app_check_token}{App\+Check\+Token}}s. This provider can be called at any time by any Firebase library that depends (optionally or otherwise) on {\ttfamily \doxylink{structfirebase_1_1app__check_1_1_app_check_token}{App\+Check\+Token}}s. This provider is responsible for determining if it can create a new token at the time of the call and returning that new token if it can.

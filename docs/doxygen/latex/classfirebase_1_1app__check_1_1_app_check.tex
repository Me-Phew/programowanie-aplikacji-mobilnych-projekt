\doxysubsection{Detailed Description}
Firebase \doxylink{classfirebase_1_1_app}{App} Check object. 

\doxylink{classfirebase_1_1_app}{App} Check helps protect your API resources from abuse by preventing unauthorized clients from accessing your backend resources.

With \doxylink{classfirebase_1_1_app}{App} Check, devices running your app will use an \doxylink{classfirebase_1_1app__check_1_1_app_check_provider}{App\+Check\+Provider} that attests to one or both of the following\+:
\begin{DoxyItemize}
\item Requests originate from your authentic app
\item Requests originate from an authentic, untampered device 
\end{DoxyItemize}

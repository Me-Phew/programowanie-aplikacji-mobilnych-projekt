Detailed Description

A Filter represents a restriction on one or more field values and can be used to refine the results of a Query.

◆ And() [1/2]

template<typename... Filters>
static Filter firebase::firestore::Filter::And ( const Filters &... filters)
inlinestatic

Creates a new filter that is a conjunction of the given filters. A conjunction filter includes a document if it satisfies all of the given filters.

If no filter is given, the composite filter is a no-op, and if only one filter is given, the composite filter has the same behavior as the underlying filter.

Parameters
[in]filtersThe filters to perform a conjunction for.
Returns
The newly created filter.

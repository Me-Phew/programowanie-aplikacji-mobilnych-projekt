Detailed Description

Child listener interface. Subclasses of this listener class can be used to receive events about changes in the child locations of a firebase::database::Query or firebase::database::DatabaseReference. Attach the listener to a location with Query::AddChildListener() or DatabaseReference::AddChildListener() and the appropriate method will be triggered when changes occur.

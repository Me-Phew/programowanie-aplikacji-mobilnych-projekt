Detailed Description

Value listener interface. Subclasses of this listener class can be used to receive events about data changes at a location. Attach the listener to a location using DatabaseReference::AddValueListener() or Query::AddValueListener(), and OnValueChanged() will be called once immediately, and again when the value changes.

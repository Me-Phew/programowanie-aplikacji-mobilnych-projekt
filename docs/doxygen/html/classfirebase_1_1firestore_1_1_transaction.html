Detailed Description

Transaction provides methods to read and write data within a transaction.

You cannot create a Transaction directly; use Firestore::RunTransaction() function instead.

Note
Firestore classes are not meant to be subclassed except for use in test mocks. Subclassing is not supported in production code and new SDK releases may break code that does so.

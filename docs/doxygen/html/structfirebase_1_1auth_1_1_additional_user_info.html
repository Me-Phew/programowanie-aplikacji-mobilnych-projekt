◆ updated_credential

Credential firebase::auth::AdditionalUserInfo::updated_credential

On a nonce-based credential link failure where the user has already linked to the provider, the Firebase auth service may provide an updated Credential. If is_valid returns true on this credential, then it may be passed to a new firebase::auth::Auth::SignInWithCredential request to sign the user in with the provider.
